# 🍽️ Restaurant Billing System (Java)

## 📌 Project Description
This is a console-based Restaurant Billing System developed using Java.  
It allows users to select menu items, enter quantity, and generate a bill with GST calculation.

## 🚀 Features
- Menu-driven interface
- Add multiple items with quantity
- Automatic bill calculation
- GST (5%) included
- Itemized receipt generation

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

## ▶️ How to Run
1. Compile:
   javac *.java

2. Run:
   java RestaurantBillingSystem
